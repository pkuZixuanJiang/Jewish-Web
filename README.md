# Jewish-Web
本项目未2024秋 北京大学通识课程《全球视野下的犹太文明》课程作业

犹太人作为世界范围内影响深远的群体之一，在科学、文化、经济和政治等领域都取得了杰出成就。然而，反犹主义也在全球范围内不断滋生和蔓延，成为当代社会的重要议题。

本项目旨在通过制作网页与问答小游戏，致力于推动公众了解犹太文化及反犹主义的社会问题。

为了方便不熟悉代码的用户，大家只需将项目文件夹下载到本地后双击start即可运行。

具体的项目介绍可参见 作品介绍.pdf
